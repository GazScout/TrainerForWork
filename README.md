Для того, чтобы открыть приложение скачайте его к себе на рабочий стол, далее откройте его папку через VS Code. Откройте терминал и пропишите команду dotnet run.

Как правило загрузка занимает около 5-10 секунд, так как используется база sqlLite и она уже вшита в проект. Далее авторизуйтесь под супер администратора: Gaz GoodLuck
